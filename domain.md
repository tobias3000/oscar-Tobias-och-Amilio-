Proxmox Server Virtual Maskin installation
Nu skall man göra en Virtual Maskin i Proxmox. Så man börjar installationen med att välja create VM och där efter så kommer det upp en sida där det står Create: Virtual Machine. Nu när insatllationen börjat skall du nu sätta in ett namn på sidan det står General och vi satt Kenny. Efter det så går man vidare till OS där du sedan väljer Use CD/DVD disc image file (iso), här väljer man sin ISO filen och du skall också laga Guest OS: till Microsoft Windows istället för Linux. Efter det går man vidare till System, här väljer man EFI Storage: där du lagar local-lvm och så avaktiverar du Add TPM:. Efter det går man vidare till Disks där du kan välja hur mycket utrymme Proxmox Virtual Maskinen skall ha och vi valde 64 GiB. Och det står på Disk size (GiB). Efter det går man vidare till CPU där man lagar Virtual Maskinen att ha ett vist antal sockets och cores. Vi valde två av båda och som blev fyra tillsamans. Efter det går man vidare till Memory där man kan välja hur mycket minne Virtual Maskinen skall ha. Vi valde 4096 Memory (MiB). Efter det går man till Networks där man far vidare till Confirm. Nu har du konfigurerat Virtual Maskinen. Nu kan du starta Virtual Maskinen som heter 100 (DC1). Efter det kommer du till en Windows server installations sida där du fyller i språket, tangetbordets inställningar, sedan väljer man installera Windows server istället för att reparera, efter det så väljer du om du har en produkt nyckel eller inte och vi har ingen så vi valde I don't have a product key, sedan väljer man vilken server installation och vi valde den som var längst nere, efter det så väljer man den disken där man vill installera på och vi valde vår 64 (GiB) och så for vi vidare och efter allt det här så börjar man Windows installationen. När Windows 11 server installationen är klar så är du färdigt att logga in på servern.
