Routerns IP address
Man börjar med att skriva in på en surfade vår routers ip addrss som var 192.168.1.1. Efter man sökt det så loggar man in med sitt användarnamn och lösenord. När man är inne på Overview så trycker man på kugghjul ikonen nere i vänstra hörnet. När man har öppnat inställningarna så väljer man Networks och där väljer man sin Default router. Nu borde det ha öppnat en panel där man sedan skall skriva in på IPv4 address sin nya IP address och där satt vi 192.168.99.1. Efter det så skall man spara det och så loggar routern ut och du skall logga in med din nya IP address. Efter det så borde du kunna komma in på routern via den nya IP addressen.

Proxmox installation
Man installationen med att installera Proxmox ISO fil via rufus på en minnessticka. Sedan efter läger man minnesstickan i servern. När man har gjort det så startar man servern och trycker på F11 på tagentbordet och det öppnar Boot Manager. När Boot Manager har öppnats så väljer man One-shot UEFI Boot Menu. Efter det väljer man att starta upp från minnesstickan och det startar upp din Proxmox installation. När du har gjort det så väljer man Graphical installation av Proxmox och går vidare. Nu accepterar man allt och går vidare. Efter det så väljer man vilken Target Harddisk du vill installera på. När du gjort det så skriver man in vilket land du befinner dig i, vilken tidszon och tangentbordets layout. Efter det så skall du skriva in vilket lösenord du vill logga in med på Proxmox och email adressen. Efter det så kommer du till IP adresserna där vi skrev in IP address som 192.168.99.2/24, Gateway 192.168.99.1, DNS Server 192.168.99.1 och Hostname wingmankenny.net och du skall sedan välja på Management interface den som har en grön boll vid sitt namn. Nu efter du har gjort det här så är det bara att börja installationen för att sedan komma in på webbsidan
