Nu borde det ha öppnat en panel där man sedan skall skriva in på IPv4 address sin nya IP address och där satt vi 192.168.99.1. Efter det så skall man spara det och så loggar routern ut dig från webbsidan och nu skall man logga in på nytt med sin nya IP address. Efter det så borde du kunna komma in på routern via den nya IP addressen som var 192.168.99.1.

Proxmox installation
När man har ändrat IP addressen skall man nu installera Proxmox på en server. Installationen börjar med att installera Proxmox ISO filen via rufus på en minnessticka. Sedan efter det läger man minnesstickan i servern. När man har gjort det så startar man servern och trycker på F11 på tagentbordet och som öppnar Boot Manager. När Boot Manager har öppnats så väljer man One-shot UEFI Boot Menu. Efter det väljer man att starta upp från minnesstickan och det startar upp din Proxmox installation. När du har gjort det så väljer man Graphical installation av Proxmox och går vidare. Nu accepterar man allt och går vidare. Efter det så väljer man vilken Target Harddisk du vill installera Proxmox på. När du gjort det så skriver man in vilket land du befinner dig i, vilken tidszon och tangentbordets layout. Efter det så skall du skriva in vilket lösenord du vill logga in med på Proxmox och email adressen. Efter det så kommer du till IP adresserna där man skriver in en Hostname, IP address, Gateway och DNS Server, vi skrev in på Hostname wingmankenny.net, IP adressen 192.168.99.2/24, Gateway 192.168.99.1, DNS Server 192.168.99.1, och du skall sedan välja på Management interface den som har en grön boll vid sitt namn. Nu efter du har gjort det här så är det bara att gå vidare till att börja installationen. När Proxmox har installerat färdigt på servern, så startar proxmox om och nu skall du se en länk på skärmen som servern är kopplad till. Länken skall se ut som det här https://192.168.99.2:8006/. Efter du ser din länk skall du nu ha en dator som du kopplar till samma nät som servern och routern. Efter du når routern med din andra dator kan du nu skriva in på webbläsare länken man ser på servern. Kom ihåg att servern skall alltid vara igång för att du skall kunna nå få något internät med dens IP address och om servern är avstängd så kommer du inte åt dens internät. Efter det skall du nu surfa på datorn länken och så kommer du att hamna på en log in sida där man skriver första som är root och andra är det lösenordet du skrev in på server installationen. Nu när du är inne på Proxmox Virtual Environment 9.1.1. Nu när man är in på webbsidan skall man nu installera en Virtual Maskin på Proxmox och du behöver en Windows 2025 server iso fil för att få igån Virtual Maskinen. När du installerat ISO filen skall på datorn skall du nu infoga den i Proxmox och för att göra det här behöver du hitta infoga ISO fil. Du hittar infoga ISO fil på Datacenter, wingmankenny och sedan skall du öppna local (wingmankenny). När du har öppnat local (wingmankenny) skall du nu välja ISO Images och där du nu infogar din Windows 2025 server ISO fil. Efter det här gör den en kort installation och nu har du en ISO fil i Proxmox. Proxmox Server Virtual Maskin installation
Nu skall man göra en Virtual Maskin i Proxmox. Man börjar installationen med att välja create VM och där efter så kommer det upp en sida där det står Create: Virtual Machine. Nu när insatllationen börjat skall du nu sätta in ett namn på sidan det står General och vi satt Kenny. Efter det så går man vidare till OS där du sedan väljer Use CD/DVD disc image file (iso), här väljer man sin ISO filen som man har laddat upp till Proxmox och nu skall du också laga Guest OS: till Microsoft Windows istället för Linux. Efter det går man vidare till System, här väljer man EFI Storage: här lagar man local-lvm och så avaktiverar du Add TPM: Efter det går man vidare till Disks där du kan välja hur mycket utrymme Proxmox Virtual Maskinen skall ha och vi valde 32 GiB. Nu har du konfigurerat Virtual Maskinen. Nu kan du starta Virtual Maskinen som heter 100 (DC1). När Virtual Maskinen har startat om kan du nu fylla i ett lösenord och användar namn åt den och efter det så är Virtual Maskinen installerad
